# 百度联盟 hao123推广 表单填写指南

## 合作形式（三选一或多选）

| 选项 | 是否勾选 | 说明 |
|------|---------|------|
| ☑️ 设hao123为首页 | ✅ 勾选 | 产品功能：可将hao123设为用户浏览器首页 |
| ☑️ 第三方浏览器合作 | ✅ 勾选 | Quick Dial 本质就是浏览器新标签页扩展 |
| ☑️ 桌面快捷方式 | ❌ 不勾 | 暂时不提供桌面快捷方式 |

## 必填字段

| 字段 | 填写内容 |
|------|---------|
| **软件名称** | Quick Dial |
| **软件描述** | 极简无广告浏览器新标签页，支持自定义导航、多搜索引擎、天气农历、壁纸主题。可作为 Chrome/Edge 浏览器扩展安装，替代原生新标签页。 |
| **软件类型** | 浏览器扩展 / 新标签页工具 |
| **下载地址** | （先占位，后面改）`https://microsoftedge.microsoft.com/addons/detail/xxx` 或 Chrome Web Store 上架后的链接 |
| **软件使用人数** | 1万以下 |
| **软件下载总量** | 1万以下 |
| **其他信息** | 浏览器扩展形式，无需安装包，用户从 Chrome Web Store / Edge Add-ons 商店直接安装。支持一键设置hao123为首页功能。 |
| **软件类型** | PC（默认已选） |

## 下载地址处理

由于表单要求提供下载地址，而 Quick Dial 是浏览器扩展（没有安装包），有两种方式：

### 方式A：先上架 Edge/Chrome 商店（推荐）
1. 先把扩展提交到 Chrome Web Store 或 Edge Add-ons
2. 审核通过后拿到商店链接填入
3. 审核时说明这是浏览器扩展，无独立安装包

### 方式B：制作一个官网下载页
1. 部署一个官网页面（如 `quick-dial.ruseo.cn`）
2. 页面包含：产品介绍 + 跳转到商店安装的按钮
3. 把这个官网地址填入下载地址

## 审核技巧

- 在"其他信息"里明确说明这是**浏览器扩展**，不是传统软件
- 强调产品功能是"新标签页"，与浏览器生态相关
- 可设置hao123为首页的功能要写清楚（勾选第一个合作形式的原因）

## 获取 tn 参数后的代码修改

审核通过后，在百度联盟后台获取 `tn` 参数，修改代码：

```typescript
// src/types/index.ts
{ 
  id: 'baidu', 
  name: '百度', 
  url: 'https://www.baidu.com/s?wd={keyword}&tn=你的tn参数' 
}
```

## 注意事项

1. 表单提交后需要 1-3 个工作日审核
2. 审核通过前，搜索分成不会生效
3. 建议同时申请 Microsoft Advertising（Bing），两个渠道不冲突
