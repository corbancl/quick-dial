## 三、盈利方向（6条路线）

### 路线 1：Pro 订阅（已有，持续优化）

当前状态：¥9.9/月 · ¥68/年 · ¥198 终身
优化方向：
- 年费/终身比例提升（年费省 43% 已够吸引力）
- 新增「家庭版」Pro（3 设备共用，¥15/月）
- Pro 用户专属壁纸库（每月更新 5 张）

### 路线 2：搜索引擎分成（零成本增收）

接入方式：
- **百度联盟**：搜索框传 tn 参数，按搜索量分成（需网站备案 + 百度联盟审核）
- **Google AdSense for Search**：国际用户 Google 搜索分成
- 不影响用户体验，无广告弹窗，搜索即分成

预估：DAU 1万时，月搜索分成约 ¥2000-5000

### 路线 3：导航位赞助（需谨慎）

- 顶部导航栏 1-2 个「推荐位」，标注「赞助」
- 收费模式：¥500/月/位
- 前提：DAU 破 5000 才有议价权
- ⚠️ 必须保持极简风格，不能变成广告站

### 路线 4：壁纸/主题市场

- 免费 12 个预设壁纸
- Pro 会员可上传自定义壁纸
- 付费壁纸包（¥6-12/套，设计师合作）
- 节日限定免费壁纸（拉活）

### 路线 5：企业定制版

- 公司内部门户页：定制 Logo、专属壁纸、预置内网链接
- 收费模式：¥1999/年/企业（含部署 + 定制）
- 目标客户：中小公司、创业团队
- 不冲突 C 端产品，独立品牌线

### 路线 6：移动端增值服务

- 小程序免费使用基本功能
- 小程序 Pro 解锁云同步 + 多设备
- 微信小程序广告组件（Banner广告，非侵入式）
- 与浏览器扩展 Pro 会员打通（买一得多端）

## 五、技术债务 & 风险

| 风险 | 应对 |
|------|------|
| Chrome 商店审核不通过 | 确保隐私政策完整、权限最小化、无远程代码执行 |
| Manifest V3 强迁移 | 已使用 V3，无需额外适配 |
| 竞品抄袭 | 开源 MIT 不防抄，但品牌 + 生态壁垒可防守 |
| 云同步服务器成本 | 当前用户量可忽略，规模化后迁移至腾讯云 COS |
| 搜索引擎分成政策变动 | 多渠道分散（百度 + Google + Bing） |
