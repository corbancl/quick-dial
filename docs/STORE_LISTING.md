### 详细说明（Detailed Description）
```
Quick Dial（呲啦起始页）是一款极简无广告的浏览器新标签页扩展。

告别臃肿的导航站和杂乱的新标签页——Quick Dial 让你的每次上网都从干净、快速、自定义的页面开始。

✨ 核心功能

📌 自定义导航
- 拖拽排序，分组管理，无限添加常用网站
- 70+ 内置 emoji 和 FontAwesome 图标，一键美化

🔍 多引擎搜索
- 内置 Google / 百度 / Bing / 搜狗 / 知乎 / B站 等 12 种搜索引擎
- Pro 版解锁更多引擎 + 自定义搜索

🌤️ 天气与农历
- 实时天气，7 日预报
- 农历日期、节气、宜忌

🎨 壁纸与主题
- 纯色 / 渐变 / 自定义图片 / 随机壁纸
- 深色 / 浅色模式，自适应切换

📊 数据统计（Pro）
- 点击排行，访问数据可视化

🔒 隐私优先
- 所有数据存储在本地浏览器
- 不上传任何个人信息
- 无广告、无追踪

---

Quick Dial — 你的浏览器，应该由你做主。
```

## 隐私政策

由于扩展不上传用户数据，使用简短声明即可：

> Quick Dial 所有数据默认存储在您浏览器的本地存储中。扩展向 api.ruseo.cn 请求天气、农历和壁纸数据。Pro 用户可选择将数据同步至 sync.ruseo.cn 云端（cilacila.cn 备案通过后切换）。

---

## 审核注意事项

1. **host_permissions**：如审核要求说明，解释为获取天气/农历 API 数据
2. **service_worker**：background.js 仅用于点击扩展图标时打开新标签页
3. **newtab override**：只替换新标签页，不修改其他页面
4. 首次提交建议不开启付费功能，审核通过后再迭代加入
