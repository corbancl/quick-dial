# 软著申请清单

## 需要准备的材料

### ✅ 1. 软件说明书
`docs/software-description.md` — 包含全部所需内容，可直接排版打印

### 📝 2. 源程序文档（需自行处理）
- 从前30页和后30页各截取源代码
- 格式：每页50行，A4纸
- 页眉标注「软件名称+版本号」
- 方法：从 `M:\new\src\` 选取核心文件打印

### 📝 3. 申请表
- 在中国版权保护中心 https://register.ccopyright.com.cn 在线填写
- 软件全称：呲啦起始页软件
- 简称：Quick Dial
- 版本号：V1.0
- 分类号：30200-0000（通用应用软件）

### 📝 4. 身份证明
- 个人申请：身份证正反面复印件

## 推荐代理机构（可选）
- 自行申请：约 ¥300 官方费用，30个工作日
- 代理机构：¥300-800 服务费，全程代办

## 源程序文档截取方法
```bash
# 合并核心源文件
cd M:\new\src
cat App.svelte types/index.ts > ../docs/source-pages.txt
# 分页：每50行一页，取前30页 + 后30页
```
