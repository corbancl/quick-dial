📌 Speed Dials

Adding Dials

  • Click the + button at bottom-right
  • Paste a URL — name and favicon are auto-detected
  • Choose from 7 icon categories with 60+ emojis + 18 brand icons
  • Custom icon URLs available in Advanced Options

Groups

  • Free: up to 3 groups. Pro: unlimited
  • Right-click a card → Edit → assign a group
  • Drag cards between groups

Bookmark Import

  • Open Import/Export → Import Browser Bookmarks
  • Select a Chrome/Edge HTML bookmarks file
  • Auto-deduplication; extras go to "Default" on free plan

🌤 Weather & Lunar

Auto-detects location via IP. Shows temperature, humidity, wind. Click to expand 3-day forecast.

Lunar calendar shows date, zodiac, solar terms. Tap to see auspicious/inauspicious directions.

🎨 Wallpaper & Themes PRO

12 free gradient presets. Pro users can upload local images (max 5MB), set URL wallpapers, or use random online wallpapers. Adjust blur (0-20px) and brightness (50%-150%). Custom CSS supported.

💾 Backup & Restore

Export all data as JSON. Import to restore. Clear all with confirmation (backup first!). Auto-saves every change.

☁️ Cloud Sync PRO

Register/login, then upload/download your config. Syncs dials, groups, settings, wallpaper, custom CSS, title, and footer.
