怎么切换搜索引擎

  • 点击搜索框左侧的搜索引擎名称（如「百度」）
  • 在下拉菜单中点击想要的引擎即可切换
  • Pro 用户可点击「添加自定义引擎」输入名称和搜索 URL

📌 导航管理

页面核心区域为快捷导航卡片，可自定义排列你最常用的网站。

添加导航

  • 点击右下角+ 按钮（或工具栏中「导入导出 → 导入浏览器书签」）
  • 粘贴网址，系统自动识别网站名称和图标
  • 可在 7 大分类（常用/社交/开发/媒体/办公/学习/品牌）中选择图标，共 60+ emoji + 18 个品牌图标
  • 支持自定义图标 URL（展开高级选项）

分组管理

  • 免费版最多创建3 个分组，Pro 版无限制
  • 右键导航卡片 → 编辑 → 选择分组进行归类
  • 拖拽卡片，悬停在分组名上可跨分组移动
  • 在分组管理面板中可排序、重命名、删除分组

拖拽排序与右键菜单

  • 长按导航卡片直接拖拽，调整排列顺序
  • 右键卡片弹出菜单，可编辑或删除
  • 选中卡片后按 Delete 删除，按 E 编辑

导入浏览器书签

  • 打开「导入导出」面板，点击「导入浏览器书签」
  • 选择 Chrome/Edge 导出的 HTML 书签文件
  • 自动按文件夹分组，已存在的不重复导入
  • 免费版超过 3 个文件夹的书签自动归入「默认收藏」

🌤 天气与农历

自动根据 IP 定位显示本地天气和农历信息。

天气功能

  • 实时显示：温度、天气描述、湿度、风向风力
  • 点击天气卡片可展开 3 日天气预报（今天/明天/后天）
  • 15 分钟自动刷新，确保数据准确

农历功能

  • 显示农历日期、干支纪年、生肖
  • 当天是节气时突出显示（绿色），否则显示下一个节气及日期
  • 节日/假日用红色高亮标记
  • 点击农历卡片可展开 黄历宜忌 + 财神/喜神/福神方位

自定义 CSS PRO

Pro 用户可在设置中写入自定义 CSS，实时注入页面（如修改字体、调整间距、隐藏元素等）。

💾 数据备份

导出备份

  • 工具栏 → 导入/导出 → 导出备份
  • 生成 JSON 文件（含导航、分组、设置、壁纸、最近访问）
  • 文件名格式：speed-dial-backup-2026-06-05.json

导入恢复

  • 选择之前导出的 JSON 备份文件，一键恢复所有数据
  • 系统自动验证文件格式，无效文件会提示

清空数据

  • 工具栏 → 导入/导出 → 清空所有数据（需要确认）
  • 此操作不可恢复，建议先导出备份

☁️ 云同步 PRO

更换设备或重装浏览器后，快速恢复你的起始页配置。

  • 注册/登录：用户名至少 2 个字符，密码至少 6 位
  • 上传到云端：点击上传按钮，数据加密传输到服务器
  • 从云端下载：在新设备登录后点击下载，3 秒后自动刷新
  • 版本冲突：云端版本更新时自动下载合并，不丢数据
  • 同步内容：导航、分组、壁纸、设置、自定义 CSS、自定义标题、自定义底部

💡 实用技巧

  • 右键菜单添加：在任意网页右键 →「添加到 Quick Dial」，自动获取网站名称和图标
  • 首次使用自动初始化：新用户自动创建默认分组和 3 个示例导航，开箱即用
  • 数据自动保存：所有改动实时保存到浏览器，无需手动操作
  • 大数据分片：超过 1MB 的数据自动分片存储，不卡浏览器
  • Pro 到期提醒：到期前 7 天，底部 PRO 徽章变红色脉冲，点击可续费
  • 新手引导：首次打开弹出 4 步引导卡片，可跳过，关闭后不再显示
